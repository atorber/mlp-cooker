# 应用模板

应用模板是 MLP Cooker 的核心功能之一,提供预定义的应用配置,支持一键部署、训练和任务创建。

## 模板结构

应用模板采用目录结构组织,每个应用包含：

- `app.json`：应用元数据
- `deploy.json`：部署模板
- `train.json`：训练模板
- `create-job.json`：任务创建模板
- `deploy-tool.json`：工具部署模板
- `{action-type}.command.sh`：命令脚本

## 使用应用模板

### 1. 浏览模板

在"应用"页面可以浏览所有可用的应用模板,支持按分类筛选。

### 2. 查看模板详情

点击应用卡片可以查看模板的详细信息,包括：
- 应用描述
- 支持的操作类型
- 配置参数
- 使用说明

### 3. 使用模板

选择相应的操作按钮（部署/训练/创建任务）,系统会自动填充模板配置,您只需要：
1. 修改必要的参数（如名称、资源配置等）
2. 点击确认即可创建

## 自定义模板

### 创建新模板

1. 在 `backend/data/app/` 目录下创建应用目录
2. 创建 `app.json` 定义应用元数据
3. 创建操作模板文件（`deploy.json`、`train.json` 等）
4. 创建命令脚本文件（`{action-type}.command.sh`）

### app.json 示例

```json
{
  "id": "my-app",
  "name": "我的应用",
  "description": "应用描述",
  "category": "推理",
  "tags": ["深度学习", "推理"],
  "actions": ["deploy", "train"]
}
```
